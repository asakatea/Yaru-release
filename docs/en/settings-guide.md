# Settings Guide

## Audience and scope
Settings controls appearance, language, security, sync behavior, and advanced integration options.

## Core categories
- Appearance and theme mode
- Language and localization
- Sync/account session controls
- Privacy and security options
- API token / integration settings

## Typical workflow
1. Set language and appearance first.
2. Verify account and sync health.
3. Configure optional integrations.
4. Re-check behavior after major updates.

## Practical tips
- Keep defaults unless you have a specific need.
- Change one advanced setting at a time and verify effects.
- Document team-level conventions for shared environments.

## Troubleshooting
- Theme mismatch: clear cached theme state and restart.
- Locale not applied: verify translation bundle version.
- Integration errors: revalidate token/base URL/provider config.
