# Getting Started

## Audience and goal
This guide is for first-time Yaru users. The goal is to complete your first learning loop in about five minutes.

## Entry points
- In-app: `Settings -> About -> Help & Docs`
- Direct route: `/docs`
- First landing screen after login: `/home`

## Quick start flow (5 minutes)
1. Sign in with email, Google, or GitHub.
2. Open **Notes** and create your first note.
3. Select one key sentence and create a flashcard.
4. Open **Memory** and complete one review rating.
5. Return to **Learning** or **Practice** to continue.

## First-day checklist
- Set language in `Settings -> Language`.
- Check sync/login status.
- Finish one note -> flashcard -> review loop.
- Bookmark this docs center for later troubleshooting.

## Permission notes
- Core learner flow (Notes/Learning/Practice/Memory/Library) only needs a normal account.
- Studio and admin pages are role-gated.

## If something looks missing
Some feature pages are still rolling out by route and role. If a menu exists but content is limited, verify account role and app version first.
