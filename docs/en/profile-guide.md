# Profile Guide

## Audience and scope
Profile pages manage personal information, social signals, and user-centric shortcuts.

## Core capabilities
- Basic profile and avatar settings
- Favorites and mistakes views
- Social stats cards and account pages

## Typical workflow
1. Open profile page.
2. Review progress and social indicators.
3. Manage favorites/mistakes.
4. Update personal information and preferences.

## Practical tips
- Use favorites for fast revisit queues.
- Treat mistakes list as your weekly remediation backlog.
- Keep profile metadata consistent across linked login methods.

## Troubleshooting
- Profile data not updated: refresh session and retry.
- Avatar upload failed: check file size/format/network.
- Empty stats: verify recent activity and sync state.
