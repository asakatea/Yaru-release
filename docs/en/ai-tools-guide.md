# AI Tools Guide

## Audience and scope
This guide explains AI-assisted features in Yaru across chat, reading, and practice scenarios.

## Core capabilities
- AI chat assistance for study questions
- Reading-side AI actions (summary/quiz/flashcards/mind map)
- Problem analysis and related reasoning support
- Tooling paths that integrate with notes and memory

## Privacy baseline
AI requests are routed through Yaru backend gateways and configuration controls. Review your settings before enabling provider-specific behavior.

## Typical workflow
1. Open the current learning artifact (note/book/problem).
2. Trigger AI action for summary or explanation.
3. Convert useful outputs into notes/cards.
4. Validate outputs before final use.

## Practical tips
- Provide concise context for better responses.
- Ask for step-by-step derivations on hard problems.
- Save reusable prompts for recurring tasks.

## Troubleshooting
- AI call failed: check network and provider config.
- Empty/invalid output: simplify prompt and retry.
- Slow response: reduce context size and attachments.
