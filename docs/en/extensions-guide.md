# Extensions Guide

## Audience and scope
Yaru extensions modules connect capture, feed, media, and automation workflows into the learning system.

## Core capability groups
- Captures: collect external snippets into your workspace
- Feeds: subscribe to structured content updates
- Media: generate or transform learning media assets
- Automation: trigger repeatable processing flows

## Typical workflow
1. Capture source content from external context.
2. Normalize and tag content.
3. Route output to notes/library/practice pipelines.
4. Track results and adjust automation rules.

## Practical tips
- Start with one narrow automation rule and expand later.
- Keep naming conventions consistent across modules.
- Review generated content before publishing/sharing.

## Troubleshooting
- Missing capture items: verify extension permissions.
- Feed lag: check polling interval/network status.
- Automation not triggered: validate trigger conditions and scope.
