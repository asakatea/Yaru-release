# User Guide

## Audience and scope
This handbook is for everyday learners using Yaru as their long-term study operating system.

## Suggested reading order
1. `getting-started`
2. `learning-guide`
3. `practice-guide`
4. `memory-guide`
5. `notes-guide`
6. `library-guide`
7. `settings-guide`

## Daily usage pattern
1. Plan tasks in Learning.
2. Practice targeted problems.
3. Capture insights in Notes.
4. Convert key points to Memory cards.
5. Review before ending the day.

## Weekly usage pattern
- Review mistakes/favorites
- Rebalance weak topics
- Clean up notes and deck structure
- Adjust settings only when needed

## Power-user tips
- Use docs center as your in-app playbook.
- Keep one stable workflow for at least 1-2 weeks before major adjustments.
- Use AI tools for acceleration, not for replacing verification.
