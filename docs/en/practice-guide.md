# Practice and Exams Guide

## Audience and scope
This guide covers Yaru's problem-bank practice flows, knowledge-point training, and exam workflows.

## Entry paths
- Main nav -> Practice
- Related exam pages from practice/exam lists

## Core capabilities
- Question-bank browsing and filtering
- Practice session creation by subject/topic/difficulty
- Knowledge-point oriented drilling
- Exam list, attempt, and review workflows
- Mistake and favorite tracking

## Typical workflow
1. Select a practice mode and filter conditions.
2. Start a session and submit answers.
3. Review explanations and mark mistakes.
4. Re-run weak topics with narrower filters.
5. Move to timed exam mode when stable.

## Practical tips
- Start broad, then narrow by weak knowledge points.
- Review mistakes on the same day while context is fresh.
- Pair practice sessions with memory reviews for retention.

## Troubleshooting
- Empty result set: broaden filters or remove strict constraints.
- Missing explanations: check locale/translation availability.
- Session state mismatch: refresh and confirm sync status.
