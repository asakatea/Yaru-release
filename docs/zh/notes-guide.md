# 笔记指南

## 模块定位与适用人群
笔记模块用于构建离线优先的个人知识库，适合长期学习沉淀、课程笔记整理、研究型写作与知识关联管理。

## 入口路径
- 主路由：`/notes`
- 数学笔记页：`/notes/math`
- 新建学习笔记入口：`/learning/notes/create`

## 功能总览
| 功能 | 说明 | 状态 |
|---|---|---|
| Vault 本地仓库 | 基于本地目录管理笔记树，支持离线优先 | 已可用 |
| 笔记树与层级管理 | 新建、重命名、删除、移动节点 | 已可用 |
| 每日笔记 | 一键打开或创建当天笔记 | 已可用 |
| WikiLink 与反向链接 | `[[...]]` 链接、自动补全、链接同步 | 已可用 |
| 知识图谱 | 图谱视图展示笔记关系 | 已可用 |
| AI 聊天面板 | 在笔记上下文内提问与辅助编辑 | 实验中 |
| 导入/导出 | Markdown 导入，Markdown/LaTeX/PDF 导出 | 已可用 |
| 闪卡联动 | 从笔记内容提取卡片并发送到记忆模块 | 已可用 |

## 典型任务流程
1. 在 `/notes` 选择或创建 Vault 目录。
2. 新建顶层主题笔记并建立子层级。
3. 使用 `[[主题名]]` 互链形成知识网络。
4. 对关键概念调用“创建闪卡”进入记忆复习。
5. 学习结束后导出 Markdown/PDF 归档分享。

## 高级用法与效率技巧
- 先建立“课程 -> 章节 -> 知识点”三层目录，再填充内容。
- 结合每日笔记记录学习日志，再用 WikiLink 连到正式主题笔记。
- 使用图谱识别孤立笔记，补全链接提高检索效率。

## 权限/角色/前置条件
- 笔记主功能面向普通登录用户。
- 使用云端同步和部分 AI 能力时，需要网络连接和有效会话。

## 常见问题与排错
- 笔记树为空：确认 Vault 是否已选择、目录是否可读写。
- WikiLink 无法跳转：检查目标标题是否存在或拼写是否一致。
- 导出失败：确认目标路径权限与磁盘可用空间。

## 相关模块联动
- 笔记 -> 编辑器指南：区块编辑与语法细节。
- 笔记 -> 记忆：知识点快速制卡复习。
- 笔记 -> 书库：阅读摘录与主题笔记互相引用。
