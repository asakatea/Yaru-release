# 学习指南

## 模块定位与适用人群
学习模块是课程驱动的学习中枢，适合按阶段推进课程、管理学习任务、结合深度资源持续精进的用户。

## 入口路径
- 学习首页：`/study`
- 学习搜索：`/learning/search`
- 课程详情：`/learning/courses/:courseId`
- 课程章节（交互课时）：`/learning/courses/:courseId/sections/:sectionId`
- 下载页：`/learning/downloads`

## 典型任务流程
1. 在 `/study` 浏览推荐和当前学习卡片。
2. 进入课程详情，按章节推进学习。
3. 需要练习时跳转 `AI 习题` 或题库练习。
4. 将任务加入 Todo，并开启番茄钟专注学习。
5. 学完后进入记忆模块做间隔复习。

## 高级用法与效率技巧
- 用“课程 + 番茄钟 + 记忆”组合构建固定日学习循环。
- 对薄弱章节先在题库做知识点定向练习，再回课程复盘。
- 借助全局搜索快速定位课程、题库、相关页面。

## 权限/角色/前置条件
- 普通课程学习功能对登录用户开放。
- LCMS 与 Studio 课程创作管理属于创作者/管理员能力。

## 常见问题与排错
- 课程打不开：确认课程 ID 有效、网络可用、账号权限正常。
- 页面内容仅为开发中提示：该页面仍在建设中（如 Review/Plan/Study Notes）。
- 推荐为空：先进行一段学习行为，系统会逐步生成个性化推荐。

## 相关模块联动
- 学习 -> 练习：课程后立即做题巩固。
- 学习 -> 记忆：将知识点转为长期记忆复习。
- 学习 -> 书库：课程主题可延伸到深度阅读资料。
