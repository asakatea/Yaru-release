# 书库指南

## 模块定位与适用人群
书库模块是阅读与知识提炼中心，适合系统阅读、章节精读、跨语种学习与阅读驱动制卡。

## 入口路径
- 书库首页：`/library`
- 图书详情：`/library/books/:bookId`
- 在线阅读：`/library/books/:bookId/read`
- 思维导图：`/library/books/:bookId/mindmap`
- 上传图书：`/library/upload`
- 阅读流桌面：`/flow-desk`
- 沉浸式阅读：`/reading/:extractId`

## 功能总览
| 功能 | 说明 | 状态 |
|---|---|---|
| 书库与分类浏览 | 分类筛选、收藏、书籍管理 | 已可用 |
| Markdown/PDF 阅读 | 支持章节阅读、搜索、导航、选择文本 | 已可用 |
| 书签与阅读进度 | 记录阅读位置并恢复阅读 | 已可用 |
| PDF 注释与 OCR | 标注、高亮、OCR 识别与术语抽取 | 已可用 |
| AI Intel 工具栏 | 翻译、思维导图、测验、闪卡生成 | 已可用 |
| YAR 资源包 | 打包/导入资源与附加学习内容 | 已可用 |
| 阅读流与沉浸阅读 | Flow Desk 队列、双栏沉浸阅读 | 已可用 |

## 典型任务流程
1. 在 `/library` 选择书籍并打开阅读。
2. 对关键段落使用 AI Intel 执行翻译/测验/闪卡。
3. 为重要页面添加书签和注释。
4. 在 `/flow-desk` 安排待读项优先级。
5. 在 `/reading/:extractId` 进入沉浸阅读并做提炼。

## 高级用法与效率技巧
- 结合 OCR 与术语抽取，快速建立章节术语表。
- 使用 AI Intel 生成测验后立即进入练习或记忆复习。
- 将阅读摘录同步到笔记，形成长期知识结构。

## 权限/角色/前置条件
- 普通阅读能力面向登录用户开放。
- AI Intel 需要可用 AI 配置与网络连接。

## 常见问题与排错
- PDF 无法加载：检查文件完整性与网络缓存状态。
- AI 功能无响应：确认 AI Provider 已在设置中启用。
- OCR 结果偏差：优先选择清晰区域并提高图像对比度。

## 相关模块联动
- 书库 -> 记忆：从阅读内容生成卡片。
- 书库 -> 笔记：将重点段落沉淀到知识仓库。
- 书库 -> 练习：以章节内容生成测验与题目训练。
