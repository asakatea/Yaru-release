# 协作功能指南

## 模块定位与适用人群
协作功能覆盖论坛与团队模块，适合讨论解题、共享资料、组织学习小组和角色化协作。

## 入口路径
- 论坛：`/forum`
- 论坛帖子详情：`/forum/thread/:threadId`
- 发帖编辑：`/forum/editor`
- 团队列表：`/teams`
- 团队详情：`/teams/:teamId`

## 功能总览
| 功能 | 说明 | 状态 |
|---|---|---|
| 论坛主页 | 板块、筛选、搜索、分页加载 | 已可用 |
| 发帖与回帖流 | 帖子编辑、详情阅读与互动 | 已可用 |
| 团队列表 | 查看我的团队并创建新团队 | 已可用 |
| 团队详情 | 成员、角色、设置分页 | 已可用 |
| 邀请与角色体系 | 团队角色、邀请接受/拒绝接口 | 已可用 |

## 典型任务流程
1. 在论坛中按板块筛选主题并参与讨论。
2. 对关键问题创建新帖并附带上下文。
3. 在团队中创建学习小组并分配角色。
4. 在团队详情页管理成员与权限。
5. 将协作沉淀内容转入笔记/课程/题库。

## 高级用法与效率技巧
- 论坛提问建议附“题目来源 + 当前思路 + 卡点位置”。
- 团队内将“管理员/贡献者/成员”分工清晰，减少沟通成本。
- 对高价值讨论可固定为笔记链接，形成可追踪知识资产。

## 权限/角色/前置条件
- 基础论坛与团队功能需登录。
- 团队管理动作（成员、角色、邀请）受团队内角色限制。

## 常见问题与排错
- 无法发帖：检查登录状态和网络。
- 看不到团队管理入口：通常是当前角色权限不足。
- 团队邀请失败：确认邀请状态与目标用户是否可见。

## 相关模块联动
- 协作 -> 笔记：讨论结论沉淀为知识条目。
- 协作 -> 练习：把论坛高频题型转为训练任务。
- 协作 -> Studio：协作后进入正式内容生产流程。
