# 设置中心指南

## 模块定位与适用人群
设置中心用于管理全局体验与系统行为，适合希望稳定同步、优化主题和提高效率的用户。

## 入口路径
- 设置首页：`/settings`
- 外观：`/settings/appearance`
- 同步：`/settings/sync`
- 语言：`/settings/language`
- AI 引擎：`/settings/ai-engine`
- 隐私：`/settings/privacy`

## 功能总览
| 功能 | 场景说明 | 状态 |
|---|---|---|
| 外观与主题 | 切换浅色/深色、配色、字体与动画 | 已可用 |
| 主题配色（Warm Ether 系） | Warm Ether、Monterey、Slate 等 palette | 已可用 |
| 同步策略 | 自动同步、冲突策略、存储配置 | 已可用 |
| 语言与地区 | 多语言切换与本地化偏好 | 已可用 |
| AI 引擎配置 | Provider、模型与 API Key 管理 | 已可用 |
| 隐私与安全 | 分析开关、加密、访问保护项 | 已可用 |
| 清除数据按钮 | UI 已有，完整清理链路仍未接入 | 开发中 |

## 典型任务流程
1. 当你首次安装完成后，先设置语言与外观。
2. 再到同步页启用自动同步并确认冲突策略。
3. 如果你要用 AI，配置 provider、模型和 key 后做连通性验证。
4. 最后在隐私页关闭不需要的统计项并检查安全设置。

## 高级用法与效率技巧
- 桌面与移动设备混用时，建议统一同步频率与冲突策略，减少版本冲突。
- 主题建议先选系统默认 palette，再按阅读场景调整为 Monterey / Slate 等。
- 先跑默认配置一周，再调整高级项（动画速度、高对比、密度等）。

> [警告] “清除数据”当前仍处于开发中，执行前请先做本地或云端备份。

## 权限/角色/前置条件
- 设置中心需要登录账号。
- 部分开发者项与 token 管理受账号状态和权限影响。

## 常见问题与排错
- 同步失败：先检查网络，再检查凭据和存储配置。
- AI 不可用：核对 provider、key 和可用模型。
- 主题切换后显示异常：重启应用并清理缓存后复查。
