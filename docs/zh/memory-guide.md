# 记忆指南

## 模块定位与适用人群
记忆模块用于长期内化知识，适合备考用户、理工科公式记忆、语言词汇积累和错题复盘。

## 入口路径
- 记忆首页：`/memory`
- 复习会话：`/memory/study`
- 卡片浏览器：`/memory/browser`
- 模板管理：`/memory/templates`
- 洞察页：`/memory/insights`
- 市场：`/market`

## 功能总览
| 功能 | 场景说明 | 状态 |
|---|---|---|
| 制卡方式（问答/双向/挖空） | 从笔记或题解快速生成可复习卡片 | 已可用 |
| 复习评分（Again/Hard/Good/Easy） | 按回忆难度更新下次复习时间 | 已可用 |
| SRS 调度 | 按遗忘趋势安排复习节奏 | 已可用 |
| 导入（Anki） | `.apkg/.colpkg` 数据迁移 | 已可用 |
| 洞察统计 | 观察复习负载与正确率趋势 | 实验中 |
| 市场牌组 | 下载社区牌组与模板 | 已可用 |

SRS 原理（通俗版）：
- 你越“费劲”想起答案，下次复习间隔越短。
- 你越“轻松”想起答案，下次复习间隔越长。
- 系统目标不是让你一直刷题，而是在“快忘记前”提醒你。

## 典型任务流程
1. 当你刚学完一章内容时，从笔记中提取 5-10 张关键卡片。
2. 进入 `复习会话` 完成当日队列并如实评分。
3. 在 `卡片浏览器` 清理低质量卡片（过长、答案歧义）。
4. 在 `洞察` 查看近 7 天复习压力，调整新卡数量。

## 高级用法与效率技巧
- 当你准备考试时，先控制“新卡”数量，优先保证“到期卡”清零。
- 把难题拆成多张小卡，比做一张超长综合卡更稳定。
- 先用默认调度策略；只有在数据稳定后再调整高级参数。

> [提示] 当你写完课程总结，立即制卡并完成一次短复习，记忆巩固效果最好。

## 权限/角色/前置条件
- 基础记忆功能需登录账号。
- 云同步与市场下载依赖可用网络和有效会话。

## 常见问题与排错
- 导入失败：确认文件是 `.apkg/.colpkg` 且文件未损坏。
- 复习队列为空：检查是否选错牌组，或今日确实无到期卡。
- 指标波动大：洞察页部分指标仍在实验迭代，建议结合复习记录核对。

## 相关模块联动
- 记忆 -> `笔记指南`：从笔记持续提取复习卡。
- 记忆 -> `练习与考试指南`：把错题转入长期复习。
- 记忆 -> `书库指南`：从阅读内容生成卡片并回流复习。
