# Yaru 入门指南

## 模块定位与适用人群
本指南面向首次接触 Yaru 的普通学习者，目标是在 5 分钟内完成第一次完整学习闭环，获得明确的上手成就感。

## 入口路径
- 文档入口：`设置 -> 关于 -> 帮助与文档` 或直接访问 `/docs`
- 首次登录后默认页：`/home`

发布版安装导向（面向普通用户）：
- Windows：从官方发布页下载安装包，安装后登录账号。
- macOS：从官方发布页下载 macOS 安装包，首次启动允许应用权限。
- Linux：从官方发布页下载对应发行版包，安装后启动并登录。
- Android：从官方分发渠道安装应用并登录。

> [提示] 安装后优先在 `设置 -> 语言` 和 `设置 -> 同步` 完成基础配置，再开始学习任务。

## 典型任务流程
1. 打开应用并登录账号。
2. 进入 `笔记` 新建第一条笔记，记录今天的学习目标。
3. 在笔记中选中关键句，创建第一张闪卡。
4. 跳转到 `记忆`，完成第一次复习评分（Again/Hard/Good/Easy）。
5. 回到 `学习` 或 `练习` 继续当天任务。

> [注意] 如果你暂时看不到某些高级页面，优先检查是否需要特定角色权限。

## 高级用法与效率技巧
- 当你当天学习内容较多时，先在 `学习` 设定任务，再用 `记忆` 做收尾复盘，可显著降低遗忘率。
- 当你找不到功能入口时，优先使用全局搜索（`/search`）而不是逐层点菜单。
- 先使用默认设置完成一轮学习闭环，再进入高级设置调整同步策略和主题细节。

## 权限/角色/前置条件
- 普通学习主线（学习/练习/记忆/书库/笔记）只需登录账号。
- `Studio` 属于创作与治理模块，受角色控制（donor/contributor/admin/owner/content_admin）。
- `Studio 用户管理/审核` 需管理员级权限。

## 常见问题与排错
- 登录后被跳转到 `/auth`：通常是会话过期，重新登录即可。
- 页面入口存在但内容为空：通常是该页面仍处于开发中。
- 新文档语言页打不开：刷新后重试，确认已拉取到最新文档版本。

## 相关模块联动
- 入门 -> `普通用户手册`：按阅读路径建立长期使用习惯。
- 入门 -> `学习指南` + `练习与考试指南`：快速进入学练主线。
- 入门 -> `记忆指南` + `笔记指南`：建立输入与内化闭环。
