# 练习与考试指南

## 模块定位与适用人群
本指南面向需要系统刷题、知识点定向训练、试卷练习与考试复盘的用户。

## 入口路径
- 练习主页：`/practice`
- 题库模式选择：`/practice/question-bank/modes`
- 知识点选择：`/practice/question-bank/knowledge-points`
- 统一练习：`/practice/question-bank/unified`
- 试卷库：`/practice/papers`
- 考试列表：`/exams`

## 功能总览
| 功能 | 说明 | 状态 |
|---|---|---|
| 题库统一练习 | 支持随机、错题、收藏、知识点等模式 | 已可用 |
| 知识点定向训练 | 选择知识点后生成针对性题目会话 | 已可用 |
| 试卷库管理 | 草稿/发布/归档试卷管理 | 已可用 |
| 试卷实战流程 | 试卷作答、计时、结果展示 | 已可用 |
| 模拟考试模式 | 题库与试卷场景支持 mock/exam 风格配置 | 已可用 |
| 试卷导出完整流程 | 导出和打开导出文件仍有 TODO 分支 | 开发中 |
| 部分 provider 细项 | 局部 API/时间统计仍在完善 | 实验中 |

## 典型任务流程
1. 从 `/practice/question-bank/modes` 选择练习模式。
2. 若选知识点模式，进入知识点选择并设定题量。
3. 在统一练习页作答并完成评分。
4. 对错题加入复习路径（错题复习或记忆模块）。
5. 需要整套训练时转到 `/practice/papers` 或 `/exams`。

## 高级用法与效率技巧
- 先做“随机练习”摸底，再切“知识点模式”补短板。
- 高频错题建议并行加入记忆模块复习队列。
- 考前冲刺优先使用试卷库定时模式模拟真实节奏。

## 权限/角色/前置条件
- 普通练习与考试功能对登录用户开放。
- 部分试卷创建/编辑能力与 Studio 创作权限关联。

## 常见问题与排错
- 练习页无题：检查筛选条件是否过严，或知识点未选中。
- 结果页数据异常：刷新当前会话并重新提交一次验证。
- 导出功能不完整：当前导出链路仍有开发中分支，建议先保存到平台内复用。

## 相关模块联动
- 练习 -> 记忆：把错题知识点转为长期复习卡片。
- 练习 -> 学习：从课程章节回到题库验证掌握度。
- 练习 -> AI 工具：结合拍题与 AI 讲解进行难题拆解。
