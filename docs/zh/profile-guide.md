# 个人中心指南

## 模块定位与适用人群
个人中心用于账号信息管理、社交展示、偏好入口聚合和身份相关操作。

## 入口路径
- 个人中心主页：`/profile`
- 编辑资料：`/profile/edit`
- 关联账号：`/profile/linked-accounts`
- 徽章：`/profile/badges`
- 捐助中心：`/profile/donation`
- 外观/学习偏好：`/profile/appearance`、`/profile/learning`
- 公共主页：`/user/:userId`

## 功能总览
| 功能 | 说明 | 状态 |
|---|---|---|
| 个人资料卡片 | 头像、昵称、标题、统计信息展示 | 已可用 |
| 社交统计 | 关注/粉丝、能力雷达、活跃热力图 | 已可用 |
| 资料编辑 | 基本信息与个性化配置 | 已可用 |
| 徽章与称号 | 展示成长与成就体系 | 已可用 |
| 游客视图 | 未登录时可访问设置与法律页 | 已可用 |
| 开发日志等附加页面 | 面向特定用户群体 | 实验中 |

## 典型任务流程
1. 在 `/profile` 查看个人卡片与统计。
2. 进入 `/profile/edit` 完成资料更新。
3. 打开 `/profile/linked-accounts` 维护账号绑定。
4. 在公共主页查看他人资料并进行社交操作。

## 高级用法与效率技巧
- 定期维护公开资料，便于团队协作和社区交流。
- 结合称号/徽章展示学习路径，提升可识别度。
- 个人页常用入口可直达设置相关页面，减少导航成本。

## 权限/角色/前置条件
- 基础资料页对登录用户开放。
- 公共主页可按访问策略向其他用户展示。

## 常见问题与排错
- 资料更新未生效：刷新页面并确认网络请求成功。
- 关注列表加载失败：通常为网络或目标用户状态问题。
- 游客看不到私有数据：符合权限设计，不属于故障。

## 相关模块联动
- 个人中心 -> 设置：偏好与隐私配置。
- 个人中心 -> 协作：论坛/团队的身份展示。
- 个人中心 -> 学习/记忆：个人成长数据反馈。
