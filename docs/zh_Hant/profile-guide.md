# 個人中心指南

## 模組定位
個人中心負責個人資訊、社交指標與使用快捷入口。

## 核心能力
- 基本資料與頭像設定
- 收藏與錯題視圖
- 社交統計卡與帳號頁面

## 典型流程
1. 開啟個人中心。
2. 查看進度與社交指標。
3. 管理收藏與錯題。
4. 更新個人資訊與偏好。

## 效率建議
- 收藏可作為快速重練清單。
- 錯題清單可作為每週回補任務。
- 多登入方式下保持個人資料一致。

## 排錯
- 資料更新失敗：刷新會話後重試。
- 頭像上傳失敗：檢查檔案大小/格式/網路。
- 統計為空：檢查近期活動與同步狀態。
