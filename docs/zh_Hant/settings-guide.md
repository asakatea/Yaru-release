# 設定中心指南

## 模組定位
設定中心管理外觀、語言、安全、同步與進階整合選項。

## 核心分類
- 外觀與主題模式
- 語言與本地化
- 同步與帳號會話
- 隱私與安全選項
- API Token / 整合配置

## 典型流程
1. 先設定語言與外觀。
2. 確認帳號與同步健康。
3. 再配置可選整合能力。
4. 重大更新後回看設定行為。

## 效率建議
- 無明確需求時優先使用預設值。
- 進階設定一次只改一項，並立即驗證。
- 團隊環境建議建立共享配置約定。

## 排錯
- 主題顯示不一致：清除主題快取後重啟。
- 語言未生效：檢查翻譯資源版本。
- 整合報錯：重新核對 token/base URL/provider。
