# 練習與考試指南

## 適用範圍
本指南覆蓋題庫練習、知識點訓練與考試流程。

## 入口路徑
- 主導航 -> 練習
- 由練習/考試列表進入詳情與作答

## 核心能力
- 題庫瀏覽與條件篩選
- 按學科/難度/主題建立練習
- 以知識點進行針對性訓練
- 考試列表、作答與回顧
- 錯題與收藏追蹤

## 典型流程
1. 選擇模式並設定篩選條件。
2. 開始練習並提交答案。
3. 回看解析並標記錯題。
4. 對薄弱知識點重練。
5. 穩定後切換到計時考試。

## 效率建議
- 先廣覆蓋，再縮小到弱項。
- 當天回看錯題，保留上下文記憶。
- 練習後接記憶複習，提升留存。

## 排錯
- 題目為空：放寬篩選條件。
- 解析缺失：檢查語系與翻譯覆蓋。
- 會話狀態不一致：刷新並確認同步。
