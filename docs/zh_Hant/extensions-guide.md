# 擴展模組指南

## 模組定位
Yaru 擴展模組將捕獲、資訊流、媒體與自動化流程接入學習系統。

## 核心能力分組
- Captures：把外部內容收集進工作區
- Feeds：訂閱結構化內容更新
- Media：生成或轉換學習媒體
- Automation：執行可重複流程

## 典型流程
1. 從外部來源捕獲內容。
2. 標準化與打標。
3. 流轉到筆記/書庫/練習管線。
4. 觀察效果並調整規則。

## 效率建議
- 先從單一小規則開始。
- 命名與標籤規範保持一致。
- 發布前人工抽查自動生成內容。

## 排錯
- 捕獲內容缺失：檢查擴展權限。
- 資訊流延遲：檢查輪詢與網路。
- 自動化未觸發：核對觸發條件與範圍。
