# 協作功能指南

## 模組定位
本指南覆蓋 Yaru 的社群與團隊協作路徑。

## 核心能力
- 論壇主題與回覆
- 團隊空間與角色管理
- 協作審閱與溝通閉環

## 典型流程
1. 進入或建立討論主題。
2. 分享資源與結構化回饋。
3. 按角色分工與追蹤責任。
4. 將後續行動落到任務路徑。

## 效率建議
- 主題標題保持明確可檢索。
- 問題、解法、審閱分階段進行。
- 用權限邊界避免誤操作。

## 排錯
- 無法發文/回覆：檢查登入與角色。
- 團隊動作不可見：檢查成員資格與角色。
- 可見範圍異常：核對公開設定與工作區範圍。
