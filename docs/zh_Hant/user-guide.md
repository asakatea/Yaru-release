# 普通用戶手冊

## 適用人群
本手冊面向一般學習者，幫你把 Yaru 用成可長期運行的學習操作系統。

## 建議閱讀順序
1. `getting-started`
2. `learning-guide`
3. `practice-guide`
4. `memory-guide`
5. `notes-guide`
6. `library-guide`
7. `settings-guide`

## 每日使用節奏
1. 在 Learning 規劃任務。
2. 在 Practice 進行針對訓練。
3. 在 Notes 記錄重點。
4. 將重點轉為 Memory 閃卡。
5. 當日收尾前完成複習。

## 每週使用節奏
- 回看錯題與收藏
- 調整弱項主題
- 整理筆記與牌組結構
- 只在必要時調整設定

## 進階建議
- 把文檔中心當作內建操作手冊。
- 固定一套流程連續使用 1-2 週再調整。
- AI 用於加速，不替代最終核對。
