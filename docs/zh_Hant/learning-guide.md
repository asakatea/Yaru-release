# 學習

學習模組是你的結構化學習中心。它將步驟式課程、生產力工具與深度學習資源整合到同一個儀表板，讓你每天都能專注於進步。

## 儀表板概覽

開啟學習模組後，儀表板會一目了然地呈現三個區塊：

- **每日進度** —— 已完成的步驟數、學習時間與連續天數摘要。
- **已註冊課程** —— 你加入的每門課程卡片，包含進度條與「繼續學習」按鈕。
- **推薦內容** —— 根據你的學習活動推薦的課程與資源。

如果你尚未註冊任何課程，儀表板會顯示友善的空狀態畫面，附有快速連結讓你瀏覽課程目錄。

## 已註冊課程與步驟式課堂

每門課程拆分為**課堂 (Lessons)**，每個課堂由一連串**步驟 (Steps)** 組成。步驟共有四種類型：

| 步驟 | 用途 |
|------|------|
| **講解 (Explain)** | 以文字、圖表或多媒體介紹概念。 |
| **範例 (Example)** | 帶你走過一個具體的解題過程。 |
| **練習 (Practice)** | 透過互動題目測驗你的理解。 |
| **總結 (Summary)** | 在進入下一課前回顧關鍵重點。 |

完成一個課堂的所有步驟後即可解鎖下一個課堂。進度會自動儲存，你隨時都能從上次中斷的地方繼續。

## 生產力工具

兩項內建工具伴隨你的課程學習：

- **待辦事項 (Todo List)** —— 建立、勾選並排序與學習相關的任務，逾期項目會自動置頂。
- **番茄鐘 (Pomodoro Timer)** —— 可自訂的專注計時器（預設 25 分鐘工作 / 5 分鐘休息）。你可以在任何課程頁面啟動它，追蹤專注學習時段。

## 深度學習資源

在已註冊課程下方，儀表板提供精選的深度學習專區：

- **維基** —— 可搜尋的公式、定理與定義參考庫。
- **課程目錄** —— 瀏覽並註冊多種學科的新課程。
- **AI 習題** —— 由 AI 產生或引導的練習題，根據你正在學習的主題量身打造。

## 課程進度追蹤

開啟任何已註冊課程，即可查看詳細進度：已完成步驟、剩餘步驟、預估完成時間，以及學習時段的視覺化時間軸。利用此檢視規劃你的學習排程，保持穩定節奏。
