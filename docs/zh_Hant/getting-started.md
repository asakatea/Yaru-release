# 入門指南

## 適用人群與目標
本指南面向第一次使用 Yaru 的學習者，目標是在 5 分鐘內完成第一個學習閉環。

## 入口路徑
- 應用內：`設定 -> 關於 -> 幫助與文檔`
- 直達路徑：`/docs`
- 登入後預設首頁：`/home`

## 5 分鐘快速流程
1. 使用郵箱、Google 或 GitHub 登入。
2. 在 **筆記** 建立第一則筆記。
3. 選取一句重點，建立第一張閃卡。
4. 進入 **記憶** 完成一次複習評分。
5. 回到 **學習** 或 **練習** 繼續任務。

## 首日檢查清單
- 先在 `設定 -> 語言` 設定語系。
- 確認登入與同步狀態正常。
- 完成一次「筆記 -> 閃卡 -> 複習」流程。
- 將文檔中心加入常用入口。

## 權限提示
- 一般學習路徑（筆記/學習/練習/記憶/書庫）只需一般帳號。
- Studio 與管理頁面受角色權限控制。

## 看起來功能缺失時
部分頁面按路由與角色分批開放。若看到入口但內容有限，先檢查帳號角色與應用版本。
