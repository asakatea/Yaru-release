# 記憶指南

## 模組定位
記憶模組負責間隔重複（FSRS）學習流程，可用於牌組管理、每日複習與長期鞏固。

## 入口路徑
- 主導航 -> `記憶`
- 由筆記/編輯器中的製卡動作跳轉

## 核心能力
- 牌組列表與詳情管理
- 基於 FSRS 的到期排程
- 複習評分（Again/Hard/Good/Easy）
- 從筆記與編輯器區塊建立卡片
- 匯入/匯出流程（含相容格式）

## 典型流程
1. 建立或開啟牌組。
2. 新增卡片（手動或由內容轉換）。
3. 開始複習並完成評分。
4. 觀察到期數與每日進度。
5. 以短時高頻方式持續複習。

## 效率建議
- 一卡一知識點，避免過載。
- 卡片正反面敘述保持一致。
- 每日短複習優於集中突擊。

## 排錯
- 沒有到期卡：檢查牌組篩選與時間設定。
- 排程異常：確認裝置時間與時區。
- 匯入內容錯亂：檢查來源格式與編碼。
