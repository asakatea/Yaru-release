# 創作工作室指南

## 模組定位
Studio 是 Yaru 的內容創作與治理工作區，面向進階創作者與管理角色。

## 核心能力
- 建立與編輯結構化資源
- 審核與發布狀態管理
- 角色受控的管理流程
- 文檔與內容質量迭代

## 角色模型
Studio 寫入操作基於權限控制。常見高權角色包含 contributor/content_admin/admin/owner（依模組而定）。

## 典型流程
1. 在 Studio 中草擬內容。
2. 檢查 metadata 與語系欄位。
3. 提交審核或按策略發布。
4. 收集回饋並迭代。

## 效率建議
- 內容 ID 與 slug 儘量保持穩定。
- 先走審核再正式發布。
- 涉及 schema 變更時同步記錄遷移說明。

## 排錯
- 寫入 403：角色權限不足。
- 草稿狀態消失：檢查保存與發布路徑。
- 內容顯示不一致：刷新快取並同步文檔元資料。
